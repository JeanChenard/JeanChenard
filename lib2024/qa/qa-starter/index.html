LIB2024- QA

  1. Qus: What are the three components of Web?

    Ans: HTML, CSS, JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or a set of files over time so that you can recall specific versions later. An example of this is GitHub.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag used to create the title of a web page is the <title> tag.

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tag used to create the title of a web page is the <p> tag.

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: I think <span> would be the odd one out as it is the only one which is an inline element.

  6. Qus: What are the tags required to create a navigation list in HTML?

    Ans: <ul> or <ol> and <li> with a class=nav are the tags required to create a navigation list in HTML.

  7. Qus: What element is used to create images? Name its attributes.

    Ans: You use <img> tag to create images. Its attributes are <src>, <class>, and <alt>.

  8. Qus Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: The <b> tag is the odd one here since it isn't a table element like the other tags are.

  9. Qus: Give an example of a color picker input.

    Ans: Hex value is a color picker who's intensity varies between 00 (which would be darkest color) and ff (which would be lightest color). For example #00005a would be a dark blue color as its values are closer to 0 then f.

  10. Qus: What is the use of name and value attributes? Give two examples.

    Ans: The name attribute specifies the name of an element. For example name="Canada" specifies the element who's named Canada. The value attribute has various roles but for the most part it is used to specify the function of a tag or element (for example for an input tag, the value attribute would specify the initial value of that element).

  11. Qus: Give an example of inline style.

    Ans: <section style="font-style: italic;" class="content"> is an example of inline style.

  12. Qus: How would you link an external style sheet? Give example.

    Ans: You link it using a link tag that goes and accesses it from your style.css page like seen in the example below: <link rel="stylesheet" type="text/css" href="css/style.css">.

  13. Qus: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: .element 1 { position: absolute;
    left: 50%;
    top: 50%;
    width: 200px;
    height: 200px;
    margin: -100px 0 0 -100px; }

  14. Qus: Create CSS keyframes block called "spin" that rotates an element.

    Ans: @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }
    }

  15. Qus: Create CSS animation block to rotate an element that has the following properties:

    • class "rotate-me"
    • It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    Ans: animation:
    spin 5s linear 5;

  16. Qus: How would you include create an alert with string "I love JavaScript" using JavaScript?

    Ans: By doing this: alert(elementId); alert("I love JavaScript") alert(this); alert(elementId+"I love JavaScript");

  17. Qus: How would you fade out an element with id "fade-me" after a delay of 300ms using JQuery?

    Ans: Like this: $('fade-me').delay(300).fadeOut(300);

  18. Qus: How would you select all elements with class "my-class" using Javascript and Jquery?

    Ans: Use the querySelectorAll("my-class") to select all classes with "my-class" in them.

  19. Qus: Give an example of input field that takes phone number with format "123-456-7890"?

    Ans: You can use either an input type="number" or input type="tel" to write a phone number in HTML. In the case of tel though it links you directly to the number you are trying to reach so use number instead if you just want it in text. For example: <input type="number" id="phone-number" name="phone-number">123-456-7890</input>
